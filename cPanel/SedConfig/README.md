<strong>SedConfig</strong>
This playbook is used to update a configuration file on all a list of servers.<br>
In this example, we are forcing HTML emails on all RoundCube clients on the shared hosting environment.<br>
In Ansible, single quotes <em>'like this'</em> has to be escaped <em>''like this''</em> so you will see this being used for the playbook to work.<br>
